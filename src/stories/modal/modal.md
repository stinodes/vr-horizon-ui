## Modal

A modal component. Pops up when its visible-prop is true, and can manage closing itself.

### Props

| Prop           | Value     | Notes                                 |
| -------------- | --------- | ------------------------------------- |
| visible        | `boolean` |                                       |
| onRequestClose | () => any | A function that should handle closing |
| title          | `string`  |                                       |
