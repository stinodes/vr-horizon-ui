## Select

A drop-down style input with auto-correct, based on [React Select V2](https://react-select.com/home).

### Props

| Prop         | Value            | Notes |
| ------------ | ---------------- | ----- |
| name         | `string`         |       |
| value        | `string`         |       |
| options      | `Object[]`       |       |
| onChange     | `(event) => any` |       |
| onFocus?     | `(event) => any` |       |
| onBlur?      | `(event) => any` |       |
| placeholder? | `string`         |       |
| disabled?    | `boolean`        |       |
| error?       | `boolean`        |
